# 📊 Dynamic Pareto Analysis – Power BI Dashboard

This project presents a dynamic Pareto Analysis dashboard built using Power BI, focusing on Visual Calculations, Field Parameters, and interactive data exploration.

---

## 🚀 Overview

The dashboard helps identify key contributors (Top Contributors) based on the Pareto principle (80/20 rule or custom threshold). It allows users to dynamically explore business performance across multiple dimensions and metrics.

---

## 🎯 Key Features

- 🔄 **Dynamic Metric Selection**  
  Switch between Sales and Profit for analysis.

- 📌 **Dimension Selector**  
  Analyze data by Product, Customer, or Country.

- 🎚️ **Interactive Pareto Threshold**  
  Adjustable threshold (e.g., 70%, 80%) to define Top Contributors.

- 📈 **Pareto Chart**
  - Column chart for metric values  
  - Cumulative % line  
  - Threshold reference line  

- 📋 **Detailed Table**
  - Rank, Cumulative Total, Individual %, Cumulative %  
  - Pareto Category classification  
  - Dynamic row highlighting  

---

## 🧠 Technical Implementation

All Pareto-related calculations are implemented using **Power BI Visual Calculations**, including:

- Rank  
- Cumulative Total  
- Individual %  
- Cumulative %  
- Pareto Category  

Traditional DAX measures are used only for base aggregations (KPIs), ensuring compliance with modern Power BI practices.
